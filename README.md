<h1>Визуализатор графа</h1>

<h2>Цель</h2>
Разработать инструмент визуализации графа зависимостей для менеджера пакетов без использования готовые средств (менеджеры пакетов, библиотеки) для получения зависимостей.

<h2>Описание проекта</h2>
<h3>Сборка и запуск</h3>
Для запуска приложения требуется установить все файлы репозитория, после чего запустить программу через главный скрипт main.py:

<code>python3 main.py</code>

<h3>Принцип работы</h3>
Работа с приложением происходит в терминале, изменения значений в конфигурационном файле и последующим запуском программы.
Приложение строит граф зависимостей пакетов, описанных в формате пакетов Rust (Cargo). Все пакеты находятся в одной папке packages. 
Затем определяется целевой пакет, для которого требуется построить граф зависимостей. 
В конце выводится дерево зависимостей требуемой глубины для целевого пакета. Формат вывода также определятся перед запуском приложения.

Изменяемые переменные:
<ol>
  <li> <code>package_name</code> - имя целевого пакета</li>
  <li> <code>package_version</code> - версия целевого пакета</li>
  <li> <code>url</code> - имя файла с целевым пакетом</li>
  <li> <code>graph_max_level</code> - максимальная глубина вывода дерева зависимостей для целевого пакета</li>
  <li> <code>mode_test_dir</code> - режим поиска целевого пакета</li>
  <li> <code>file_name_graph</code> - название файла с выводимым графом</li>
  <li> <code>mode_output</code> - режим вывода графа</li>  
</ol>

Поддерживается 2 режима поиска целевого пакета: PACKAGE_NAME
<ol>
  <li> <code>PACKAGE_NAME</code> - поиск происходит по имени и версии пакета. Если не указана версия (параметр <code>package_version</code> пустой или равен символу ' '), то ищется последняя версия пакета с заданным именем. В данном режиме параметр <code>package_url</code> не используется</li>
  <li> <code>PACKAGE_URL</code> - поиск происходит по имени файла, в котором записан пакет. В данном режиме параметры <code>package_name</code> и <code>package_version</code> не используются
</ol>

Поддерживается 2 режима вывода дерева зависимостей для пакета:
<ol>
  <li> ASCII - режим вывода дерева зависимостей в виде ASCII дерева в терминал</li>
  <li> Любое другое значение - режим вывода дерева зависимостей в виде графа, построенного через Graphviz, в отдельный pdf-файл</li>
</ol>
